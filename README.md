# Parkpal

Parkpal is an original SwiftUI iOS concept app for saving a parking spot, tracking parking expiration, and navigating back to the saved location.

## Highlights
- Save the current parking location with CoreLocation
- Show the saved spot in a rounded map card with MapKit
- Track parking expiration with an animated circular countdown
- Schedule a local notification before expiration
- Launch Apple Maps navigation back to the parked car
- Store parking spots locally with Core Data
- Offer StoreKit-based upgrades for multiple spots and custom alerts

## Project structure
- `Parkpal/` contains the SwiftUI app source
- `Parkpal/Parkpal.xcdatamodeld/` contains the Core Data model
- `Parkpal.xcodeproj/` contains the Xcode project file

## Notes
This implementation is intentionally original and does not copy UI, branding, or assets from existing parking or mapping apps.
